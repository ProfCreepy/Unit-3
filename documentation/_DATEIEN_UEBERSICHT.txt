💡 TIPPS & TRICKS
─────────────────────────────────────────────────────────────────────────────

🎨 Farbe ändern?
   → Öffne src/theme.js
   → Ändere Hex-Code
   → Speichern → Alle Icons update sich automatisch!

🧩 Neuen Icon hinzufügen?
   → theme.js: Farbe definieren
   → Component: getIconColor('myIcon') nutzen
   → Import aus lucide-react

📱 Mobile testen?
   → npm run dev
   → http://IP:5173 im selben WLAN

⚡ Performance debuggen?
   → React DevTools Chrome Extension
   → Prüfe Re-Renders

🔍 CSS debuggen?
   → F12 → Inspector
   → Prüfe computed Styles
   → Check CSS Custom Properties

🐛 HÄUFIGE PROBLEME & LÖSUNGEN
─────────────────────────────────────────────────────────────────────────────

Problem:           Icons haben keine Farbe
Lösung:            ✓ theme.js in src/ ?
                   ✓ getIconColor() importiert ?
                   ✓ Cache geleert ? (Ctrl+Shift+R)

Problem:           Sidebars funktionieren nicht
Lösung:            ✓ Settings/Extensions.jsx kopiert ?
                   ✓ sidebar.css vorhanden ?
                   ✓ onClick Handler in Button ?

Problem:           CSS stimmt nicht
Lösung:            ✓ App.css vollständig ersetzt ?
                   ✓ sidebar.css hinzugefügt ?
                   ✓ npm install ausgeführt ?

Problem:           Module Not Found
Lösung:            ✓ npm install
                   ✓ npm run dev neustart
